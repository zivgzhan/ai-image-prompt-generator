Privacy Policy

This page explains how AI Image Prompt Generator handles basic privacy and analytics information.

Overview

AI Image Prompt Generator is a simple website that lets users enter image ideas and generate English image prompts for AI image generation tools.

Accounts and payments

This website does not require login, does not create user accounts, and does not collect payment information.

Analytics

This website uses Microsoft Clarity to understand how visitors use the site, such as clicks, scrolling, and page visits. This information helps improve the website experience.

What you should not enter

Please do not enter sensitive personal information, private identifiers, passwords, payment details, medical information, or other confidential data into the prompt generator.

Future updates

If the website changes or new features are added, this Privacy Policy may be updated.
